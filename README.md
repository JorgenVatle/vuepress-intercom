# VuePress-Intercom
A simple wrapper for [`vue-intercom`](https://www.npmjs.com/package/vue-intercom) that plugs into VuePress.

## Installation  
```bash
npm install vuepress-intercom
```

## Setup
> .vuepress/config.js
```js
module.exports = {
    // ...
    plugins: {
        'vuepress-intercom': {
            appId: 'your-app-id'
        },
    }
    // ...
}
```
